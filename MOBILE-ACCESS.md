# 手机访问说明

如果你需要公网访问，而不是局域网访问，请看：

- [GitHub Pages 公网部署](./GITHUB-PAGES.md)

1. 在电脑上双击运行 `start-mobile-server.cmd`。
2. 保持这个命令窗口不要关闭。
3. 让手机和电脑连接到同一个 Wi-Fi。
4. 在手机浏览器中打开窗口里显示的地址，例如 `http://192.168.1.23:8080/`。

补充说明：

- 首次运行时，Windows 可能弹出防火墙提示，请允许“专用网络”访问。
- 默认端口是 `8080`。
- 如果想改端口，可以运行：

```powershell
.\start-mobile-server.ps1 -Port 9000
```

- 停止服务时，在命令窗口按 `Ctrl + C`。
